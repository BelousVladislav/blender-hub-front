Воркерами являються особи які надають свої обчислювальні ресурси в короткочасну оренду клієнтам. За таку послугу воркери можуть отримати винагороду в виді донатів через вказані ним сервіси або криптогаманці.

Для того щоб стати воркером необхідно:

  • Стабільне підключення до інтернету;
  • ПК або ноутбук, що має дискретну відеокарту;
  • Програмне забезпечення воркера;
  • Встановлений Node.js;
  • Програма Blender;
  • Встановлений Redis;
  • WinRar архіватор;

Почнемо з програмного забезпечення воркера.

Завантажити програмне забезпечення воркера можливо за посиланням: FreeBlender Worker

Натисніть Code -> Download ZIP та розархівуйте архів в зручне для вас місце.

Далі, встановлення Node.js.

Для завантаження Node.js необхідно перейти за посиланням: Download Node.js та написнути кнопку Download Node.js

Далі на черзі програма Blender.

Для завантаження Blender необхідно перейти за посиланням: Download Blender та написнути кнопку Download Blender

Завантажену програму встановити за замовчуванням на комп'ютер.

Далі, встановлення Redis.

Для завантаження та встановлення Redis необхідно перейти за посиланням: Download Redis та виконувати вказівки заявлені на офіційному сайті.

Далі, завантаження архіватора WinRar.

Для завантаження WinRar необхідно перейти за посиланням: Download WinRar та написнути кнопку Download WinRar

WinRar повинен бути встановлений за шляхом C:\Program Files\WinRAR

Повертаємось до раніше встановленого програмного забезпечення воркера.

В папці проекту знаходиться файл .env. Його потрібно відкрити на редагування.

Вміст файлу виглядає наступним чином:

PORT=6060
BLENDER_PATH=C:\Program Files\Blender Foundation\Blender 4.0\blender
WORKERUUID=89DBA036-7BAD-43F3-93CD-2A0553CF6D94
REDIS_HOST=localhost
REDIS_PORT=6379
CENTRAL_SERVER_KEY=http://213.231.7.96:9000/api/

Необхідно змінити дві властивості BLENDER_PATH та WORKERUUID.

BLENDER_PATH - це шлях виконуючого файлу blender.exe на вашому комп'ютері.

WORKERUUID - це унікальний ідентифікатор клієнта, який під'єднує воркер до системи FreeBlender.

Знайти цей ідентифікатор можна в додатку на сторінці "Профіль" -> "Мої дані"

Після зміни цих властивослей, файл необхідно зберегти та закрити.

Для запуска воркера необхідно з термінала Windows перейти в папку з раніше завантаженим програмним забезпеченням воркера та виконати наступні команди:

  • npm install та натиснути Enter на клавіатурі.
  • node ./dist/main.js та натиснути Enter на клавіатурі.

Якщо останній рядок має вигляд: Worker registered! то воркер працює та вже очікує завдання для рендерингу.

Веб-додаток воркера

Воркер має свій веб-додаток, який доступний за адресою: http://127.0.0.1:6060/dashboard

Додаток відображає черги та завдання, що згруповані за статусом. Кожне завдання містить інформацію про час початку та завершення виконання завдання, а також веде журнал з інформацією про стан завдання або помилки які сталися під час його виконання.
