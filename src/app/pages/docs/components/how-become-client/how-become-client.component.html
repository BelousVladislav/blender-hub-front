З свого профілю ви можете сворити особистий проект, перейшовши за посиланням "Профіль" -> "Мої проекти" -> "Створити проект"

"UUID Project Token" генерується при створенні проекту, та надає змогу отримати доступ до нього напряму з Blender за допомогою адона FreeBlender. Цей ключ важливо зберегти, так як він більше не буде з'являтись.

Після створення проекту можна одразу перейти до його карточки, натиснувши на ім'я проекту.

Проект виглядає зовсім пустим, тому давайте перейдем на сторінку "Рендеринг", та передамо наш перший blend файл на рендеринг. Для цього натискаєм на кнопку Новий рендеринг

Тут все просто! Завантажуємо потрібний нам .blend чи .rar(якщо є допоміжні матеріали) файл на форму. Починаєм рендеринг натиснувши на кнопку Почати рендеринг

Далі магія! Файл відправляється до вільного воркера, який розпочинає рендеринг проекту. Результатом є .jpeg файл, який можна завантажити собі.

  • Підтримується лише файли формату .rar;
  • Архів повинен містити папку з такою ж назвою як і сам архів. В самій папці повинні бути blend файл та допоміжні матеріали проекту;
  • Допоміжні матеріали проекту повинні мати відносний шлях;

На вкладці "Налаштування" ви маєте змогу:

  • Редагувати інформацію проекту;
  • Видалити проект
  • Згенерувати новий "UUID Project Token"

Адон "FreeBlender" дозволяє вам відправляти файл на рендеринг напряму з "Blender". Для встановлення адону перейдіть за посиланням: Завантажити FreeBlederAddon

Натисніть Code -> Download ZIP

Розархівуйте архів в зручне для вас місце.

Далі відкрийте програму Blender та перейдіть в налаштування(Edit -> Preferences...)

Перейдіть на вкладку Add-ons та написніть кнопку Install... Оберіть раніше завантаженний файл та натисніть Install Add-on

В результаті цих дій адон буде встановлено в Blender

Для відправки файлу на рендеринг, необхідно вказати ProjectId проекту та UUID Project Token(він згенерувався під час створення проекту)

ProjectId відображається на вкладці проекту "Інфо" -> "Про проект"

Якщо ви забули UUID Project Token, згенеруйте новий. Сторінка проекту -> "Налаштування" -> Generate

Тепер під час моделювання в Blender варто лише зберегти файл та натиснути кнопку Send To FreeBlender.

Файл буде передано на рендеринг, а ви можете продовжувати працювати над моделлю.
